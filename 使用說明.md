# AA 創作翻譯輔助小工具 — 使用說明

本工具用於將含有 ASCII Art (AA) 圖像的日文漫畫翻譯成中文，並在替換文字時自動補齊空白，避免 AA 圖排版崩壞。

---

## 基本工作流程

1. 貼上或從網址讀取日文原文
2. 點「提取日文」，取出需翻譯的文字
3. 將提取結果進行翻譯(AI、手動皆可)
4. 將翻譯貼回「填入翻譯」區
5. 點「替換翻譯並編輯」，進入編輯器微調後儲存

---

## 主畫面

### 頂部工具列

| 按鈕 | 說明 |
|------|------|
| ⚙ | 開啟全域設定（自動複製、歷史記錄數量、暫存管理等） |
| 批次搜尋 | 在資料夾內的所有 HTML 檔中搜尋與取代 |
| 編輯模式 | 切回目前已開啟的編輯畫面（避免誤按返回後需重跑流程） |
| 📖 Wiki 對照 | 從 Wiki 頁面抓取角色名稱的日文↔中文對照，供複製到術語表 |
| 📥 讀取設定 | 從 `AA_Settings.json` 讀入過濾規則與術語表 |
| 📤 儲存設定 | 將目前的過濾規則與術語表存入 `AA_Settings.json` |
| 🔧 提取Debug | 對選取的文字逐步顯示提取流程的分析報告，用於排查提取結果異常 |

### 原始文本區

貼上日文原文（含 AA 圖）的地方。

| 按鈕 / 欄位 | 說明 |
|-------------|------|
| 🌐 網址讀取 | 開啟網址讀取視窗，直接從網頁抓取原文 |
| ◀ 上一話 | 自動讀取上一話的網頁內容（需先用網址讀取過） |
| 下一話 ▶ | 自動讀取下一話的網頁內容 |
| 📋 複製網址 | 複製目前讀取的網址到剪貼簿 |
| 標題欄位 | 輸入作品標題（選填），用於產生 HTML 的檔名 |
| 🕘 | 顯示最近使用的作品 / 作者歷史記錄，可快速填回 |
| 話數欄位 | 輸入話數（如 `1`），用於 HTML 檔名；貼上文字時若偵測到「第N話」會自動填入 |

#### 網址讀取視窗

| 按鈕 / 功能 | 說明 |
|-------------|------|
| 讀取（綠色） | 輸入網址後點此抓取，或按 Enter 觸發 |
| 忽略留言 | 勾選後只保留作者本人的貼文，排除回文 |
| 作者名稱欄 | 指定作者名稱；若留空則自動偵測第一個發文者 |
| 關聯記事列表 | 同系列各話的連結，可直接點擊切換，▶ 標記為當前話 |
| 上一話 / 下一話 | 快速讀取相鄰話 |
| 讀取紀錄 | 最近 50 筆紀錄，點擊標題直接重新讀取；複製按鈕可複製網址 |
| 清除紀錄（紅色） | 清空所有讀取紀錄 |

---

### 右側面板

#### 自訂過濾規則

每行一條規則（支援正則表達式）。提取日文時若文字符合任一規則，該行會被排除不提取。

#### 術語表

格式為 `日文原文=中文替代`，每行一條。替換翻譯時會自動套用。

| 元件 | 說明 |
|------|------|
| 一般 | 長期使用的術語（儲存至 `AA_Settings.json`） |
| 臨時 | 本次使用的臨時術語（同樣儲存至設定檔） |
| 跳到重複（橘色） | 若術語表中出現重複的原文，點此可依序跳至每一處 |

> 兩張術語表合併後一起套用，翻譯時不需手動切換。

---

### 提取區

#### ⬇️ 提取日文 ⬇️

核心按鈕。點擊後系統分析原始文本，過濾掉 AA 圖形，提取出需要翻譯的純文字片段，以 `001-1|日文` 格式顯示於「提取結果」區。若「提取後自動複製」開啟（⚙ 設定中），提取後自動複製至剪貼簿。

#### 提取結果區

| 按鈕 | 說明 |
|------|------|
| 複製全部 | 複製所有提取結果到剪貼簿 |
| 複製上半 | 只複製上半部分（長文分段交給 AI 翻譯時使用） |
| 複製下半 | 只複製下半部分 |
| 加入自訂過濾 | 將選取的提取結果加入右側「自訂過濾規則」，下次提取時排除 |

---

### 填入翻譯區

將 AI 回傳的翻譯結果貼在此處。格式應保持 `001-1|中文翻譯`，每行一條，與提取結果對應。貼上後系統會自動檢查格式是否異常並顯示警告。

---

### 底部 Action Bar

| 按鈕 | 說明 |
|------|------|
| 🚀 替換翻譯並編輯 | 執行替換，套用術語表並自動補空白，然後進入 HTML 編輯器 |
| 💾 替換翻譯並儲存 | 執行替換後直接彈出另存新檔對話框，不進編輯器 |
| 📥 讀入暫存 | 恢復上次關閉前的編輯內容（如有） |
| 📂 打開已儲存的 HTML | 開啟既有的 HTML 檔，直接進入編輯器繼續修改 |

## 編輯器

進入編輯器後可對翻譯結果進行手動微調。

### 工具列 — 全文替換

| 按鈕 / 欄位 | 說明 |
|-------------|------|
| 原文欄 / 翻譯欄 + 執行 | 即時術語替換：填入一組原文與替代文字後點執行，立即套用至全文（含自動補空白） |
| 重套術語 | 根據目前術語表對全文再次執行替換（若術語表有更新可用此重新套用） |

### 工具列 — 選區操作

先用滑鼠選取文字，再點以下按鈕：

| 按鈕 | 說明 |
|------|------|
| 上色 | 為選取文字加上顏色標記（`<span style="color:...">`）；若已有上色則改為去色 |
| 🎨 | 選擇要使用的顏色 |
| 消空白 | 刪除選取文字內所有空白 |
| 補空白 | 在選取文字的每個字元之間插入兩個全形空白 |
| 對話框修正 | 對選取的對話框（普通框 / 吶喊框 / 斜線框 / 方框）重新對齊邊框 |
| 對齊上一行 | 在游標處補空白，將游標後第一個非空白字元對齊到上一行末端 |
| 自動判斷 `Alt+Q` | 依選取狀態自動決定動作：選多行 → 對話框修正；選單行 → 上色；無選取 → 對齊上一行 |
| 對話框(全) | 掃描全文，自動偵測並修正所有對話框 |

### 工具列 — 視窗控制

| 按鈕 | 說明 |
|------|------|
| 底色 | 調整編輯區底色（僅供編輯時視覺參考，不影響輸出） |
| 💾 儲存 | 彈出另存新檔對話框（`Ctrl+S` 則直接覆寫原檔） |
| 📂 開啟 | 開啟另一份已儲存的 HTML 檔 |
| ← 返回 | 回到主畫面（或批次搜尋） |
| 關閉 | 關閉視窗 |

### 搜尋列（`Ctrl+F` 開啟）

| 按鈕 | 說明 |
|------|------|
| 搜尋欄 | 輸入關鍵字，不分大小寫搜尋 |
| 下一個 | 找下一個符合的位置（循環） |
| 🎲 1D10:10 | 快速搜尋殘留的骰子格式（提取時應已自動去除，可用此確認） |
| ✕ | 關閉搜尋列（或按 `Esc`） |

## 編輯器快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `Alt+1` | 回到編輯模式（若在比對或預覽模式中） |
| `Alt+2` | 切換「原文比對」模式（可與翻譯後的內容對照，詳見下方說明） |
| `Alt+3` | 切換「上色預覽」模式（模擬瀏覽器顯示顏色效果） |
| `Alt+Q` | 自動判斷並執行對話框修正 / 上色 / 對齊上一行 |
| `Alt+C` | 將選取範圍提取日文後複製到剪貼簿（不修改文件） |
| `Alt+E` | 將選取範圍內的術語替換結果還原回日文原文 |
| `Alt+W` | 以原文同位置的內容覆蓋目前選取範圍 |
| `Ctrl+F` | 開啟 / 關閉搜尋列 |
| `Ctrl+S` | 直接覆寫存檔（若為新檔則走另存） |
| `Esc` | 關閉搜尋列（若已開啟），否則返回主畫面 |

### 關於「原文比對」（Alt+2）

按 `Alt+2` 可切換至原文比對模式，左側顯示翻譯後內容、右側顯示翻譯前的日文原文，方便對照確認。

**原文是「按下翻譯按鈕當下，原始文本欄的內容」**，系統會自動儲存，下次開啟同一份 HTML 時仍可使用。

原文在以下時機寫入：

| 時機 | 說明 |
|------|------|
| 按「🚀 替換翻譯並編輯」 | 進入編輯器時自動儲存 |
| 按「💾 替換翻譯並儲存」 | 存檔完成時自動儲存 |
| 在編輯器中按 `Ctrl+S` 或「💾 儲存」 | 存檔時自動儲存 |

原文以 HTML 檔名為索引，存放於 `aa_original_cache.json`。下次用「打開已儲存的 HTML」或從批次搜尋開啟同一份檔案時，系統會自動載入對應的原文供比對。

> **若原文顯示空白**，表示該 HTML 是在這些時機之外產生（例如從其他地方複製過來），或從未用本工具執行過翻譯，因此沒有原文記錄可載入。

## 批次搜尋

點工具列「批次搜尋」開啟。可在指定資料夾內的所有 HTML 中搜尋關鍵字或正則表達式，並進行批次取代。

| 功能 | 說明 |
|------|------|
| 搜尋 | 在資料夾所有 HTML 的內文中搜尋，最多顯示 500 筆結果 |
| 替換（單筆） | 對單一結果執行取代 |
| 全部替換 | 對所有搜尋結果執行批次取代 |
| 全部復原 | 將本次批次替換涉及的所有檔案一次還原 |
| 復原（單筆） | 還原單一檔案的修改 |
| ✕（排除） | 將某筆結果從批次取代範圍中移除 |
| 開啟 | 在編輯器中開啟該檔案，並跳至對應行 |
| 快速替換面板（右側） | 輸入 `A=B`，點 `[→]` 按鈕快速填入搜尋 / 替換欄；雙擊可加入主術語表 |

## 資料保存說明

| 檔案 | 用途 |
|------|------|
| `AA_Settings.json` | 過濾規則與術語表（手動讀取 / 儲存） |
| `aa_settings_cache.json` | 程式狀態暫存（每次輸入後自動儲存，關閉重開不丟失） |
| `aa_original_cache.json` | 原文暫存（用於編輯器中的原文比對功能） |
| `aa_crash.log` | 程式異常時的錯誤記錄 |
